# Documentación de savr

savr es una aplicación de presupuesto de base cero que te ayuda a darle un propósito a cada unidad de tu dinero antes de gastarlo. Esta guía cubre todo lo que necesitas para configurar tus cuentas, armar un presupuesto funcional y usar las funciones de savr con confianza.

## El método de presupuesto de base cero

La idea central: **ingresos menos egresos es igual a cero.** Cada unidad de dinero que ganas se asigna a una categoría — comida, renta, ahorros, entretenimiento — hasta que no queda nada sin asignar. Cuando gastas, el saldo de la categoría disminuye. Si te excedes, mueves dinero de otra categoría.

Este método funciona porque te obliga a tomar decisiones de gasto con anticipación, no después del hecho. El resultado: en todo momento sabes exactamente qué tienes disponible para todo lo que te importa.

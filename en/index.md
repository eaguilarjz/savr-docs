# savr Documentation

savr is a zero-based budgeting app that helps you give every unit of money a job before you spend it. This guide covers everything you need to set up your accounts, build a working budget, and use savr's features confidently.

## The zero-based budgeting method

The core idea: **income minus outgo equals zero.** Every dollar you earn is assigned to a category — groceries, rent, savings, fun — until nothing is left unallocated. When you spend, the category balance decreases. If you overspend, you move money from somewhere else.

This method works because it forces you to make spending decisions in advance, not after the fact. The result: at any moment, you know exactly what's available for everything you care about.
